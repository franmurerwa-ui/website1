# Discover Kigali — Rwanda's Shining Capital

A five-page informational website about things to do and places to eat around Kigali, Rwanda. Built as part of the SheCanCode Web Development curriculum (Assignment 2).

---

## Project Description

Discover Kigali is a fully responsive, multi-page website that serves as a travel guide to Rwanda's capital city. The site covers top activities, restaurants, cultural experiences, and nature destinations — all accessible from Kigali. It demonstrates proficiency in semantic HTML5, CSS layout techniques (Flexbox and Grid), form design, media embedding, and responsive web design.

---

## HTML Pages Included

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with overview, quick facts table, lists, video, map, newsletter form |
| Activities | `activities.html` | Things to do in Kigali — memorial, arts, biking, gondola, walking tours |
| Restaurants | `restaurants.html` | Top places to eat — fine dining, street food, restaurant comparison table |
| Culture | `culture.html` | Arts, markets, music, dance, cultural calendar table |
| Nature | `nature.html` | National parks, gorilla trekking, safari, wildlife tips |

---

## CSS Files

Each HTML page has its own dedicated CSS file:

- `index.css` — Home page styles and design system tokens
- `activities.css` — Activity cards, schedule table, booking form
- `restaurants.css` — Restaurant grid, comparison table, reservation form
- `culture.css` — Culture cards, event calendar, tour request form
- `nature.css` — Nature cards, parks comparison table, safari booking form

## What This Project Demonstrates

### HTML Skills
- Proper use of semantic elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<figure>`
- Forms with `<input>`, `<select>`, `<textarea>`, `<label>`, `<button>`
- Tables with `<caption>`, `<thead>`, `<tbody>`, `<th>`, `<td>`
- Ordered lists (`<ol>`) and unordered lists (`<ul>`) on every page
- `<video>` and `<audio>` elements with controls and fallback content
- Embedded Google Maps via `<iframe>`
- Internal links between all pages and external links to relevant resources
- At least 10 images across all pages using `<img>` with descriptive `alt` attributes
- Metadata: `<meta charset>`, `<meta viewport>`, `<meta description>`, `<title>`

### CSS Skills
- Separate CSS file per page with full design system (custom properties / CSS variables)
- Flexbox for navigation, card layouts, footer, and form rows
- CSS Grid for destination cards, tips sections, and the nature featured card
- Hover effects on all navigation links, cards, buttons, and images
- Smooth transitions and micro-animations (`transform`, `transition`, `@keyframes`)
- CSS custom properties (variables) for consistent theming
- Responsive design via `@media` queries for mobile, tablet, and desktop
- Image styling: object-fit, border-radius, box-shadow, overlay effects
- Form styling with focus states and visual feedback
